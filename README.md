# kkGame
Play 2 kkbox html widget at same time

## Sample

#### embed a song

```
/?song=${trackId}
```

#### autoplay a song

```
/?song=${trackId}&autoplay=true
```

#### embed 2 song

```
/?song=${trackId1}&song=${trackId2}
```

#### autoplay 2 song

```
/?song=${trackId1}&song=${trackId2}&autoplay=true
```
